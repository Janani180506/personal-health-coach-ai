Project Documentation
1️⃣ Project Title

Personal Health Coach AI Agent

2️⃣ Problem Statement

Many users provide long-term medical history and daily wellness data repeatedly to AI systems. This increases token usage, processing cost, and context overload.

3️⃣ Objective

To build a Health Memory Compression AI Agent that:

Converts long medical history into structured health summaries

Reduces repeated token usage

Provides personalized recommendations efficiently

4️⃣ System Architecture

User Input (Medical History + Wellness Data)
⬇
Memory Compression Module
⬇
Structured Health State Vector
⬇
AI Recommendation Engine
⬇
Personalized Health Advice

5️⃣ Technologies Used

Python

Flask (Backend)

HTML, CSS, JavaScript (Frontend)

OpenAI API / LLM

Git & GitHub

6️⃣ Installation Steps
git clone https://github.com/Janani180506/personal-health-coach-ai.git

pip install -r requirements.txt
